Rain final product folder
=========================

Use this folder when you only want to test the built app.

Current testable builds:
- Rain-release-android.apk
- Rain-windows-portable\rain.exe
- Rain-windows-portable.zip
- Rain-openrelay-demo-windows-portable\rain.exe
- Rain-openrelay-demo-windows-portable.zip

Archive:
- archive\previous-final-product
- archive\test-extract
- archive\test-run

The archive folder only keeps old or extracted build copies so the project root
does not get crowded again.
