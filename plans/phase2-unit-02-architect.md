# Unit 02 — Architect Interfaces
Goal: Define system boundaries, data models, and API contracts between Backend, Frontend, and DB agents.

Deliverables:
- Data model definitions (Plan, TaskUnit)
- API contracts (endpoints, payload schemas)
- Interface sketches for inter-agent communication

Done Criteria:
- Interfaces documented and reviewed
- No ambiguous types in the interfaces
- A small, testable contract artifact exists (e.g., TypeScript types)

Risks:
- Mismatched contracts across teams; mitigate with explicit versioning and review.
